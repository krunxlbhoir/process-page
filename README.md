# Process Page

A simple 3-step process page built with HTML and CSS.

## Demo
https://krunxlbhoir.github.io/process-page/

## How to run locally
Open index.html in any web browser.

## Notes
- Uses ::before pseudo-element for heading highlight
- Box-shadow: 0px 30px 100px rgba(17, 23, 41, 0.05)
- Fully responsive using media queries
- Includes icons and dashed separators
